Introduction

Pour créer un site web, on doit donner des instructions à l'ordinateur. Il ne suffit pas simplement de taper le texte qui devra figurer dans le site (comme on le ferait dans un traitement de texte Word, par exemple), il faut aussi indiquer où placer ce texte, insérer des images, faire des liens entre les pages, etc.

Définition html

Le HyperText Markup Language, généralement abrégé HTML ou dans sa dernière version HTML5, est le langage de balisage conçu pour représenter les pages web.
Ce langage permet : d’écrire de l’hypertexte, d’où son nom, de structurer sémantiquement la page, de mettre en forme le contenu, de créer des formulaires de saisie.

Définition balise

Une balise HTML est un élément de base du système de codage HTML utilisé, en grande partie, pour la création d'une page web.
Ce sont les balises html qui permettent de hiérarchiser et de structurer le contenu d'une page Web.

1- Les balises de base

Les balises de premier niveau sont les principales balises qui structurent une page HTML.

Conclusion

Le html est le langage utilisé pour écrire code dans un éditeur de texte dans le but de créer un site web.
Son rôle est de gérer et d'organiser le contenu qui doit etre affiché sur la page web (texte, lien, image, etc)
Le html5 est la derniere version qui inclue facilement des vidéos, avec un meilleur agencement du contenu, de nouvelles fonctionnalités pour les formulaires, etc
